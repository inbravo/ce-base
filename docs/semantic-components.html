Seller type: 3P → processing = 2 days

Weekend in window → +2 days

Shall I find the same item from an Amazon-dispatched seller? That would arrive July 4th — before your deadline.

FieldTypeRule
order_idSTRINGNOT NULL · must exist in orders table
status_codeENUMone of: DEL / ATT / SAF / NEI / LIT
scan_timestampTIMESTAMPNOT NULL · must be ≤ current time
photo_urlSTRINGNOT NULL when status_code = DEL
carrier_nameSTRINGNOT NULL
Quality Rules
▸ Status must update within 30 minutes of carrier scan
▸ photo_url must resolve (HTTP 200) within 5 minutes of DEL status being set
▸ No status may revert to a prior state — DEL cannot go back to ATT
SLA: 99.9% uptime · ALERT: quality breach → auto-escalate to on-call within 5 minutes · BREAKING CHANGE: requires sign-off
